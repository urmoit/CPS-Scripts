Clear Water (Island) Script

Description:

Purpose:

This script automates the process of removing water in a grid-based environment, such as a game map. It efficiently identifies and clears all water tiles within a specified range, ensuring the area is fully dry.

Important Note:

The script will only function if you are in /ghost mode. To activate ghost mode, type /ghost in the chat before running the script.


--[Clear water island]
--[Turn on /ghost]
            
--[Main Script]
for x = 0, 199, 1 do
    for y = 199, 0, -1 do
        tile = GetTile(x,y)
        if tile.flags.water == true then
            SendPacketRaw(false, {state = 32, x = x * 32, y = y * 32})
            Sleep(1)
            while GetTile(x,y).flags.water ~= false do
                SendPacketRaw(false, {type = 3, value = 822, px = x, py = y, x = x * 32, y = y * 32})
                Sleep(400)
            end
        end
    end
end
LogToConsole("DONE")
