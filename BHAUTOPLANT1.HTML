Auto Plant Script

Description:

This script automates the process of planting seeds across a grid-based environment in Growtopia. It efficiently handles seed planting by identifying the optimal locations and ensuring the seeds are planted correctly across the entire area. This tool is ideal for players looking to automate the tedious task of planting large numbers of seeds, helping them save time and focus on other aspects of the game.

Delay = 100

for y = 186, 0, -1 do -- THE 186 IS THE LOCATION OF THE FIRST ROW OF PLATFORMS In the tutorial video you can find the the location.
    for x = 0, 199 do
        local tile = GetTile(x, y)
        if tile.fg == 0 and GetTile(x, y + 1).fg ~= 0 then
            FindPath(x, y, Delay)
            SendPacketRaw(false, {px = x, py = y, x = x*32, y = y*32, type = 3, value = 5640})
            
            Sleep(20)
        end
    end
end
