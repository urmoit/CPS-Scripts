Auto Harvest Script

Description:

This script automates the process of harvesting Trees across a grid-based environment in Growtopia. It efficiently handles tree harvesting by identifying the optimal locations and ensuring the trees are harvested correctly across the entire area. This tool is ideal for players looking to automate the tedious task of harvesting large numbers of trees, helping them save time and focus on other aspects of the game.

Delay = 100

for y = 186, 0, -1 do -- THE 186 IS THE LOCATION OF THE FIRST ROW OF PLATFORMS In the tutorial video you can find the the location.
    for x = 0, 199 do
        local tile = GetTile(x, y)
        if tile and tile.extra and tile.extra.progress and tile.extra.progress == 1.0 then
            FindPath(x, y, Delay)
            SendPacketRaw(false, {py = y, px = x, type = 3, x = x*32, y = y*32, value = 18})
            Sleep(20)
        end
    end
end
